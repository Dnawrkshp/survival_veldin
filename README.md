# survival_veldin

Veldin custom map for [Horizon's Zombie Survival mode](https://github.com/Horizon-Private-Server/horizon-deadlocked-patch/tree/develop/survival).

![thumbnail](HUD/thumb.png)

## Dependencies

1. Docker
2. [Horizon Forge](https://github.com/Horizon-Private-Server/horizon-forge/wiki/Setup-Guide)
3. Git LFS

## Install Guide

1. Clone the repository into your Forge `/Assets/Maps/` folder. Make sure that the folder name matches the respective `.unity` file in the repo.
2. Open the `.unity` scene file in Forge.

For example:
```sh
cd ~/horizon-forge/Assets/Maps
git clone https://github.com/Dnawrkshp/survival_veldin.git survival_veldin
```
